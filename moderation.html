🔢 Liste des mobs

Assigne un numéro ordre à chaque mob. Les mobs sans numéro s'affichent en dernier par ordre alphabétique. Les changements sont sauvegardés directement dans Firestore.

Chargement…
⚔️ Ordre des items

Assigne un numéro ordre à chaque item. Glisse-dépose ou entre un numéro de position. Les changements sont sauvegardés directement dans Firestore.

Chargement…
🔗 IDs fantômes

IDs référencés dans des crafts ou des drops de mobs, mais sans item défini dans la base Firestore.
Cliquer Créer ouvre le creator avec l'ID pré-rempli.

Chargement…
🧑 Ordre des PNJ

Glisse-dépose les régions pour changer leur ordre. Les PNJ d'une région sont triés alphabétiquement. La sauvegarde met à jour l'ordre des régions et des PNJ dans Firestore.

Chargement…
🗺️ Régions orphelines

Détecte les mobs avec une région inconnue (référence cassée) et les régions sans aucun mob.

Chargement…
👾 Mobs orphelins

Détecte les mobs sans région.

Chargement…
📋 Quêtes orphelines

Quêtes sans mapId — non reliées à un marqueur sur la carte interactive.

Chargement…
📍 Ordre des régions

Glisse-dépose les régions pour définir leur ordre d'apparition dans le jeu. L'ordre est utilisé dans le creator pour le choix de région.

Chargement…
📜 Liste des quêtes

Glisse-dépose les quêtes pour définir leur ordre d'apparition dans le wiki. Les changements sont sauvegardés directement dans Firestore.

Chargement…
🎛️ Contrôles Creator

Active ou désactive les outils visibles dans le Creator. Les changements sont appliqués en temps réel pour tous les contributeurs.
